# Calculator
calculator made with tkinter <br/>
made from this tutorial https://youtu.be/QZPv1y2znZo

## Quick Start
1. cloning. <br /> 
`$ git clone https://github.com/devincapriola/Calculator.git`
2. running the calculator. <br /> 
`$ pyrthon calculator.py`
